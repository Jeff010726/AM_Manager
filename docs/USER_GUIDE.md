# 使用文档

## 1. 登录
- 使用账号密码登录。
- 默认账号：
  - 管理员：admin@example.com / admin123
  - 访客：visitor@example.com / visitor123

## 2. 权限说明
- `admin`：可进行全部操作（主数据、库存、项目、用户）。
- `visitor`：默认只读；若被加入项目，可提交该项目 Commit 更新状态。

## 3. 主数据管理
- 在“主数据”页创建分类与产品 SKU。
- SKU 字段包含：编码、名称、分类、单位、安全库存等。

## 4. 库存管理
- 在“库存”页查看完整口径：
  - 总库存
  - 在途
  - 在库
  - 可用
  - 预留
  - 消耗
  - 安全库存
  - 缺口
- 管理员可执行：入库、在途登记、在途入库、预留、消耗等。

## 5. 项目协作
- 在“项目”页查看项目列表。
- 管理员可把用户分配进项目并设置项目角色。
- 项目成员可提交 Commit（标题、说明、状态变化、进度）。
- 最新 Commit 会驱动项目最新状态。

## 6. 关键规则
- 项目消耗必须基于预留单。
- 不能绕过预留直接项目消耗。
- 库存写操作使用幂等键防止重复提交。

## 7. 常见问题
- 401：Token 过期或无效，请重新登录。
- 403：权限不足（比如 visitor 调写接口）。
- 409：库存不足或幂等键重复。
